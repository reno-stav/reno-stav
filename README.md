# Elkar

https://elkar.github.io/

fresh clone:

* requires Ruby, Bundler & NodeJS with Grunt

```
git clone ...
cd elkar
gem install bundler
bundle install

npm install
```


Server lunch

```
# first terminal window:
grunt serve

# second terminal window
grunt
```
